Migration
=========

.. _migration:

Snooze 2.0 is the Go rewrite of the Python 1.x server. This section
collects everything an operator needs to move an existing 1.x deployment
to 2.0 without re-bootstrapping the database.

.. toctree::
    :titlesonly:

    python-to-go

The migration guide's canonical source is
``docs/migration/python-to-go.md`` (Markdown). The RST page above is a
short companion that links out to it.
