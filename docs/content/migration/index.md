# Migration

Snooze 2.0 is the Go rewrite of the Python 1.x server. This section collects everything an operator needs to move an existing 1.x deployment to 2.0 without re-bootstrapping the database.



See the [Migrating from Snooze 1.x to 2.0](./python-to-go.md) guide for the field-by-field upgrade path.
